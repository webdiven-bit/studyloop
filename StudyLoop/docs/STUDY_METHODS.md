# Study Methods Used in StudyLoop

## Active Recall
- Questions generated from your own materials
- Answers hidden by default

## Spaced Repetition (Planned)
- Topics you struggle with reappear more often

## Mnemonics & Music Memory
- If you select *Music* as a preference, difficult concepts are turned into short rhythmic phrases
- If you select *Writing*, the app generates short memorable sentences to copy down

## Metacognition
- You see accuracy and weak areas after each session
